# Agent

## 정의
현재 이 문서에서 agent 는 Cursor CLI 기반으로 동작하는 작업 수행용 worker AI agent 를 의미한다.

## 역할
- 문서 안에서 agent 의 의미를 고정한다.
- 사람 역할과 AI 역할을 구분한다.
- 자동 실행 주체를 명확히 한다.
- 다른 종류의 agent 와 혼동되지 않게 한다.

## 규칙
- 여기서 agent 는 사람을 뜻하지 않는다.
- 여기서 agent 는 범용 AI 전체를 뜻하지 않는다.
- 현재 문맥에서는 Cursor CLI 위에서 실행되는 worker 만 뜻한다.
- 문서 전반에서 같은 의미로 일관되게 사용한다.

## 한 줄 암기법
agent 는 "Cursor CLI 기반으로 실제 작업을 수행하는 worker AI"다.
